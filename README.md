# signos-xml-parser
Zodiac Finder, ou "Descubra seu Signo"

Aplicação Web desenvolvida para identificar o signo do zodíaco com base na data de nascimento inserida pelo usuário.

## Tecnologias e Ferramentas
- **Linguagem:** PHP
- **Estruturação de Dados:** XML 
- **Frontend:** HTML5, CSS3 e Bootstrap 5
- **Servidor Local:** XAMPP

## Funcionalidades
- **Interface Intuitiva:** Formulário responsivo construído com Bootstrap.
- **Manipulação de Dados:** Leitura e processamento de arquivos XML através da função `simplexml_load_file` do PHP.
- **Lógica de Backend:** Algoritmo que realiza o parsing das datas e compara intervalos para retornar o signo correspondente de forma dinâmica.

## Como executar o projeto
1. Clone este repositório na pasta `htdocs` do seu servidor XAMPP.
2. Inicie os módulos Apache e MySQL no painel do XAMPP.
3. Acesse `localhost/nome-da-pasta/index.php` no seu navegador.
